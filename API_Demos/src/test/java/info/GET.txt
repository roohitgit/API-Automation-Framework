List of all objects


Description
Retrieves a predefined set of sample objects from the public API. 
This endpoint provides a limited, reserved selection available for demonstration purposes - not all stored objects created 
by you or others. However, if you want to access multiple objects that you’ve created, you can use query parameters to 
retrieve specific objects by their IDs. Alternatively, you can sign up and use the authenticated API to access objects 
from your own collections in a more convenient and confidential manner

Parameters
• id (string[]) - Optional query parameter for specifying one or more object IDs. When provided, 
the response includes only the specified objects and excludes all others. Supports multiple values by repeating 
the parameter in the query string (e.g., ?id=3&id=5&id=10)
